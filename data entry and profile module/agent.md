# Agent

you will create a data entry browser module linked to a profile page. The goal is to link it later to a browser app. 
The profile page will ask the name and first name of a collector. 
Age, location, mail address are optional fields. 
Then the collector will have to point out how he wonts to feed the data to the data entry module. 

he can upload his artworks manually one by one, he can upload his artworks with a file (pdf) or whether he wants to give you the link to an online platform so that the data entry module can scrape it. 
the data entry module will select one picture, the name of the artwork and the name of the artist and list them. 

In the POC we will scrape this website : https://www.2dgalleries.com/ en meer specifiek voor het profiel Jan : https://www.2dgalleries.com/profile/jan. 

als je gevraagd wordt om via de data entry module https://www.2dgalleries.com/profile/jan te scrapen, dan zou je dus 57 foto's moeten vinden voor 57 verschillende artworks.
